Spice Equivalents fREEDA translates the elements specified in the Spice netlist into fREEDA native elements.
Level Translation for Spice Translation scheme for relating MODELS and LEVELS in Spice syntax to native fREEDA elements

The catalog description of an element can be obtained using "freeda -c elementName" This can be useful as it is very easy to add elements ot fREDEA. The description is taken directly from the internal data structure, with links to documentation provided by the element model developer.
