f REEDA

In fREEDA Each element is represented by its own model (and code and documentation).  This contrasts with SPICE which uses a combination of the first letter of the element name, the type of the model (specified in the .MODEL statement) and the LEVEL parameter to identify an element. The translation is done in the parser.
